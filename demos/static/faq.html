FAQ

How do I start OpenVila?

Run openvila init, then openvila scan, then openvila run.

Can users run actions directly?

No. Users can request actions, but each request must be approved by the owner.

Can I bring my own model endpoint?

Yes. Set LLM_ENDPOINT and LLM_API_KEY in your environment.

Is multilingual Q&A supported?

Yes. The assistant can answer in the user's language based on the knowledge source.
